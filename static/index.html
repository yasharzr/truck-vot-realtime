The Problem

Highway 401 through Toronto is the busiest highway in North America. Trucks passing through Toronto — not stopping — can bypass it via Highway 407 ETR, a tolled express route. But is the toll worth the time saved? This dashboard answers that question in real time.

The Model

Based on a Mixed Logit (MXL) discrete choice model estimated from a stated preference survey of Ontario truck drivers. The model accounts for taste heterogeneity — different drivers value time differently. Key finding: the average truck driver's Value of Time is $81.01/hr (SD $10.09).

Market VOT (no model needed)

The "market VOT" is what Highway 407 effectively charges per hour of time saved: toll cost / (time saved in hours). This is pure math from real-time data — no model assumptions. When the market VOT exceeds $81/hr, the toll route is overpriced for the average truck.

Data Sources + Limitations

Travel times: Google Maps Directions API (real-time, every 3 min). Tolls: 407 ETR 2026 rate chart for heavy multi-unit vehicles (Class 4, transponder). Choice probability: MXL6 model from thesis SP data — a baseline estimate; real choices may differ. The driver survey collects new SP-off-RP data to improve future models.

Citation: Zarrin Zadeh, Y. (2021). Truck Route Choice Behaviour and Value of Time: Highway 401 vs 407 ETR Corridor. MASc Thesis, York University, Toronto.
